---

## Add the Marketplace Source

Run the following command in Claude Code to register the plugin source:

```bash
/plugin marketplace add obra/superpowers-marketplace
```

## Install the Plugin

Then install the plugin from the registered source:

```bash
/plugin install superpowers@superpowers-marketplace
```
